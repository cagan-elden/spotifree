## What is Spotifree?
Since spotify has a bunch of annoying advertisements every two songs and buying premium is useless the Spotifree is being built. Spotifree is a react-native based mobile application which gets the songs the user wants through web-scrapping. It is also an application which can be used offline by downloading your favourite playlists. The app doesn't have an account system because I believe that everybody has their right to keep their data for themselves.
